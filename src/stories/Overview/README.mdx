---

## Why tokens + Tailwind v4
- We use `@theme` to map tokens ⟶ Tailwind utilities.  
- We wipe Tailwind’s default palette to **enforce** semantic colors.

```css
/* foundations.css */
@import "tailwindcss";

@theme inline { --color-*: initial; }      /* lock down defaults */

@theme static {
  --color-bg: hsl(var(--brand-bg));
  --color-text: hsl(var(--brand-text));
  --color-genre-drama: hsl(var(--genre-drama));
  /* ...all tokens used by utilities... */
}
```

## Theme switching

The app/Storybook toggles `<html data-theme="dark|light">`.
Tokens declare overrides like:

```css
/* tokens.css */
:root { --brand-bg: 210 35% 6%; /* ... */ }
[data-theme="light"] { --brand-bg: 0 0% 100%; }
```
